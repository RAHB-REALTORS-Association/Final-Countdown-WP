A simple countdown block for the WordPress block editor.

== Description ==

A simple countdown block for the WordPress block editor that allows you to set a date and time for the countdown to end.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/final-countdown-block` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the "Plugins" screen in WordPress
3. Insert the "Final Countdown" block into your post or page.
